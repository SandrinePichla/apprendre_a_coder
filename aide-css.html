Où inserer le code CSS

Créer un lien vers un fichier externe "Style.CSS"

  • Il s'agit d'un fichier avec l'extention .CSS et non .html
  • On va écrire le code CSS à l'interieur de ce fichier CSS. Il s'appliquera à tous les fichiers qui contiendront la balise <link rel="stylesheet" href="../CSSstyle/Style.CSS">
  • La meilleure solution, car lorsqu'on veut changer le code du fichier CSS, qui pilote le code de tous les fichiers liés
  • Lorsque l'on veut donner un style différent à une portion de texte, on insère l'attribut class= et dans le CSS, mettre.nom
  • On peut également utiliser l'id (#) à la place de class, mais attention, on ne peut avoir q'une seule foisréférencer qu'un seul id dans la page. Du coup, dans le CSS mettre # à la place de .
  • On peut utiliser l'étoile * au lieu du point, toutes les balises seront alors concernées
  • Pour selectionner par exemple, le premier paragraphe sous le titre h1, on note h1 + p
  • Pour selectionner par exemple, tous les paragraphes sous le titre h2, on note h2 p

Ecrire directement dans l'en-tête head

  • Utiliser la balise <Style> </Style>
  • A l'interieur de cette balise, mettre le code CSS

Ecrire dans une balise

  • Inserer à l'interieur d'une balise l'attribut Style="Color: blue"
  • Exemple, dans un fichier, dans une balise <p Style="Color: blue">Texte du paragraphe</p>
  • Très peu utilisé, car lorsqu'on veut modifier, il faut aller sur chaque fichier, et sur chaque attribut

Comment fonctionne le code CSS

  • Pas de chevrons mais des accolades
  • On indique la balise mais pas entre chevrons. Juste p par exemple. Mais on peut mettre plusieurs balises à la suite exemple, p,h1,h3
  • Puis on ouvre l'accolade, et on écrit des attributs exemple color="", font-weight=""...
  • puis on ferme l'accolade
  • CSS = cascading style sheets = cascade. Si l'enfant n'a pas de style, il est hérité des parents. Pour une même balise, le dernier style retenu sera celui écrit en dernier. Pour une balise identique celle qui a des attributs sera plus précise, on conserve son style

Formater le texte

  • La taille : font-size (16px, 1.3em, ou 110 %)
  • La hauteur de la ligne : line-height= 1.4;
  • Espace netre le lettres : letter-spacing=0.2
  • La police : font-family (arial, verdana, sans-serif)
    1. pour en télécharger et pour l'inserer : Le fichier de police (ici MaNouvellePolice.eot) doit ici être situé dans le même dossier que le fichier CSS :
      2.                         @font-face {
                            font-family: 'MaNouvellePolice';
                            src: url('MaNouvellePolice.eot') format('eot'),
                                url('MaNouvellePolice.woff') format('woff'),
                                url('MaNouvellePolice.ttf') format('truetype'),
                                url('MaNouvellePolice.svg') format('svg');
                        }
    2. Italique (font-style: italic), gras (font-weight: bold), souligné (text-decoration: underline et text-decoration-style/ line / color ) , barré (text-decoration: line-through)
    3. Alignement = text-align: right; left; justify, center. Seulement pour les élements en display inline ou inline-block. Pas pour les blocks
    4. Flottant = float: left; right
    5. Couleur du texte = color: maroon; ou color: #FFFFFF;
    6. Couleur du fond = background-color: black; sur la balise body
    7. Couper les mots trop longs overflow-wrap, hyphens=manual et écrire '& shy;' à l'endroit où il faudrait couper, (ou word-break)

  • bordures et des ombres
    1. Bordure à un élément : propriété border. Il faut indiquer la largeur de la bordure, sa couleur et son type (trait continu, pointillés…).
      2. p { border-left: 2px solid black;
                            border-right: 2px solid black;}
    2. On peut arrondir les bordures avec border-radius : forme de l'arrondi pour chaque coin
      3. p{border-radius: 10px 5px 10px 5px; }
    3. On peut ajouter une ombre aux blocs de texte avec box-shadow. On doit indiquer le décalage vertical et horizontal de l'ombre, son niveau d'adoucissement et sa couleur.
      4. p{box-shadow: 6px 6px 6px black;}
    4. Le texte peut lui aussi avoir une ombre avec text-shadow
      5. p{text-shadow: 2px 2px 4px black;}
  • apparences dynamiques
    1. Au survol a:hover, p:hover
    2. Au clic :active et lors de la sélection :focus
    3. Lorsque le lien a déjà été consulté :visited

unités de mesures

  • px
  • % (largeur : % du bloc parent. Pour la hauteur, ça ne fonctionne pas)
  • vh (1% de la hauteur de l'espace visible)
  • vw (1% de la largeur de l'espace visible); vmin et vmax(hueteur ou largeur facteur limitant)
  • em (proportionnel à la taille de la police du bloc)
  • rem (proportionnel à la pocice définie dans la racine = body ou html, toutes les hauterus seront identiques dans tous les blocs)

Formater des blocs

  • Dimension : width = largeur, height = hauteur. On peut réduire la taille ex : 50%, ou donner une taille 250px. Des tailles mini (min-width) ou max (max-height)
  • Les marges. Interieures (padding) et exterieures (margin). Une seule valeur = s'applique aux 4 côtés du bloc; sinon préciser margin-bottom / padding-left...
    •                 p{width: 350px; /* On a indiqué une largeur (obligatoire) */
                margin: auto; /* On peut donc demander à ce que le bloc soit centré avec auto */
                border: 1px solid black;
                text-align: justify;
                padding: 12px;
                margin-bottom: 20px;}
  • Lorsque le texte ne tient pas dans le bloc défini, utiliser overflow, ce qui crée des barres de défilement
    •                 p {width: 250px;
                height: 110px;
                text-align: justify;
                border: 1px solid black;
                overflow: auto; }
  • Lorsque le mot est trop long (ex URL) on peut forcer la coupure avec word-wrap: break-word

FLEXBOX

  • Tuto Flexbox sur OCR
  • Définir un conteneur avec plusieurs éléments (div)
    •                         < div id="conteneur">
                        <div class="element 1">Element  </div>
                        <div class="element 2">Element </div>
                        <div class="element 3">Element </div>
                        </div>
  • Flex : les blocs se placent à côte
    •                         #conteneur{display: flex;}
  • Donner une direction : display: flex; flex-direction: column;
    • row : organisés sur une ligne (par défaut)
    • column : organisés sur une colonne
    • row-reverse : organisés sur une lignemais en ordre inversé
    • column-reverse : organisés sur une colonne, mais en ordre inversé.
  • Retour à la ligne : display: flex; flex-wrap: wrap;
    • nowrap : pas de retour à la ligne (par défaut)
    • wrap : les éléments vont à la ligne lorsqu'il n'y a plus la place ;
    • wrap-reverse : les éléments vont à la ligne, lorsqu'il n'y a plus la place, en sens inverse.
  • Alignement sur axe principal à l'horizontal : display: flex; justify-content
    • flex-start : alignés au début (par défaut)
    • flex-end : alignés à la fin
    • center : alignés au centre
    • space-between : les éléments sont étirés sur tout l'axe (il y a de l'espace entre eux)
    • space-around : idem, les éléments sont étirés sur tout l'axe, mais ils laissent aussi de l'espace sur les extrémités.
  • Alignement sur axe principal à la vertical : display: flex; flex-direction: column; justify-content: center
  • Alignement axe secondaire : display: flex; justify-content: center; align-items: center
    • stretch : les éléments sont étirés sur tout l'axe (valeur par défaut)
    • flex-start : alignés au début
    • flex-end : alignés à la fin
    • center : alignés au centre ;
    • baseline : alignés sur la ligne de base (semblable à flex-start)
  • Avec flex-wrap , on peut autoriser les éléments à revenir à la ligne s'ils n'ont plus d'espace.
  • S'il y a plusieurs lignes, on peut indiquer comment les lignes doivent se répartir entre elles avec align-content
  • Chaque élément peut être réagencé en CSS avec order
  • Avec la super-propriété flex , on peut autoriser nos éléments à occuper plus ou moins d'espace restant.

Avant FLEXBOX (à éviter, pour les vieux sites)

  • Float Exemple : nav {float: left;} par exemple, mais il faut mettre une marge à la prochaine section {margin-left: 170px}
  • Display: inline-block à associer avec vertical-align: top par exemple
  • {Position: absolute ou fixed ou relative } absolute (où l'on veut sur la page, préciser l'endroit et les px), fixed (toujours visible) et relative (permet de décaller)
