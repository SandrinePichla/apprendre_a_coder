Où inserer le code CSS

Créer un lien vers un fichier externe "Style.CSS"

  • Il s'agit d'un fichier avec l'extention .CSS et non .html
  • On va écrire le code CSS à l'interieur de ce fichier CSS. Il s'appliquera à tous les fichiers qui contiendront la balise <link rel="stylesheet" href="../CSSstyle/Style.CSS">
  • La meilleure solution, car lorsqu'on veut changer le code du fichier CSS, qui pilote le code de tous les fichiers liés
  • Lorsque l'on veut donner un style différent à une portion de texte, on insère l'attribut class= et dans le CSS, mettre.nom
  • On peut également utiliser l'id (#) à la place de class, mais attention, on ne peut avoir q'une seule foisréférencer qu'un seul id dans la page. Du coup, dans le CSS mettre # à la place de .
  • On peut utiliser l'étoile * au lieu du point, toutes les balises seront alors concernées
  • Pour selectionner par exemple, le premier paragraphe sous le titre h1, on note h1 + p
  • Pour selectionner par exemple, tous les paragraphes sous le titre h2, on note h2 p

Ecrire directement dans l'en-tête head

  • Utiliser la balise <Style> </Style>
  • A l'interieur de cette balise, mettre le code CSS

Ecrire dans une balise

  • Inserer à l'interieur d'une balise l'attribut Style="Color: blue"
  • Exemple, dans un fichier, dans une balise <p Style="Color: blue">Texte du paragraphe</p>
  • Très peu utilisé, car lorsqu'on veut modifier, il faut aller sur chaque fichier, et sur chaque attribut

Comment fonctionne le code CSS

  • Pas de chevrons mais des accolades
  • On indique la balise mais pas entre chevrons. Juste p par exemple. Mais on peut mettre plusieurs balises à la suite exemple, p,h1,h3
  • Puis on ouvre l'accolade, et on écrit des attributs exemple color="", font-weight=""...
  • puis on ferme l'accolade

Balises

  • Saisir un commentaire (aide pour le dev, non visible sur la page : /* Commentaire */
