MANUEL PHP en français

A quoi sert le PHP : à génèrer du code HTML, c'est un langage de programmation

Installer php

  • Ouvrir visual studio code
  • Installer WAMP
  • Créer un fichier php : "index.php" dans sites\learn\stm\app\scripts\php
  • Démarrer wamp dans le dossier où se trouve le fichier php : configurer MAMP / préférences/ Web Server/ selectionner le dossier parent
  • Une balise PHP a la forme <?php /* Code PHP */ ?>
  • Configurer PHP pour afficher les erreurs. Modifier le fichier appelé php.ini, avec error_reporting E_ALL et display_errors On. En cas d'erreur, elle s'affichera sur la page web

Débuter

  • Exemple avec une instruction echo : (permet d'insérer du texte dans la page web)

    • <?php
      echo "Exécution de PHP";
      ?>
    • Ouvrir chrome et taper http://localhost/php/ où php est le nom du dossier où se trouve le fichier php
    • Sur la page web, on pourra lire Exécution de PHP

  • On peut mettre des balises PHP n'importe où dans du code HTML, même à l'interieur d'une balise HTML, exemple : < meta < ?php /* Code PHP */ ? > charset="utf-8" />
  • Lorsqu'on regarde le code source, on ne voit pas qu'il y a du code PHP
  • Grande importance de noter des commentaires, si une ligne // et texte, si plusieurs lignes /* texte et */

Les variables

  • Info le temps de générer la page web = éphémère

  • Consituée d'un nom et d'une valeur : ex : Nom = âge / valeur = 50

  • Ecrire les variables selon la convention camelCase, toujours démarrer par $

  • Différent types de valeur

    • String = chaine de caractères (=texte) Ex <?php $fullname = "Toto"; $email = 'toto@exemple.com'; ?> faire attention aux " " et ' '.
      Utiliser \ pour bloquer l'action des "" et ' ' lorsque ce n'est pas du code. Ex $variable = "Mon \"nom\" est Toto"; $variable = 'Je m\'appelle Toto';
    • Int = Nombre entier Ex <?php $userAge = 50; ?>
    • Float = Nombres décimaux Ex <?php $price = 57.3; ?> (attention . pas ,)
    • Bool = valeur booléenne (vrai ou faux) Ex <?php $isAuthor = true; $isAdministrator = false; ?>
    • NULL = ne contient rien Ex <?php <?php $noValue = NULL; ?>

  • Concatener : Ecrire du texte autour d'une variable :

    • On peut écrire les variables à l'intérieur de guillemets doubles <?php $fullname = "Toto"; echo "Bonjour $fullname et bienvenue sur le site !"; ?>
    • Il faut écrire les variables à l'intérieur de guillemets simples <?php $fullname = 'Toto'; echo 'Bonjour ' . $fullname . ' et bienvenue sur le site !'; ?>

  • Conditions = c'est ce qui va réellement vous permettre de créer une application dynamique
    • if
    • else et elseif
    • Exemple : $isAllowedToEnter = "Non"; if ($isAllowedToEnter == "Oui") {echo "Vous êtes autorisé";} elseif ($isAllowedToEnter == "Non") {echo "Accès refusé ";} else {echo "erreur de choix ";}
    • si on veut que la condition soit fausse, on écrit : if (!$isAllowedToEnter). Ca revient à écrire if ($isAllowedToEnter == false)
    • switch (il ne peut tester que l'égalité)
    • case
  • Faire des calculs
    • On peut écrire : $number = 2 + 4; echo ' $number = 2 + 4 = ' . $number ; le résultat s'affiche
    • Des calculs simples : $number = (1 + 2) * 2;
    • des fonctions : $number = 10; $result = ($number + 5) * $number;
    • Des modulo (reste d'une division) $number = 10 % 3;
  • Symboles et mots clés à connaitre
    • * Pour le égal, pour une valeur exactement égale à on mettra === * Or : altGr + 6

  • Exemple : if (($isEnabled && $isOwner) || $isAdmin) {echo 'Accès validé';} else {echo 'Accès refusé';}
  • les ternaires : des conditions condensées. Ils servent à faire deux choses sur une seule ligne :
    • Tester la valeur d'une variable dans une condition
    • Affecter une valeur à une variable selon que la condition est vraie ou non
