Caractères réservés

  • Le & s'écrit & a m p ;
  • Le chevron ouvert < s'écrit & l t ;
  • Le chevron fermé > s'écrit & g t ;
  • La quote "s'écrit & q u o t ;

    • Eléments HTML

  • < h1 > < /h1> TITRE DE PARAGRAPHE NIVEAU 1
  • < p> < /p> PARAGRAPHE
  • < strong> < /strong> GRAS
  • < em>< /em> ITALIQUE
  • < br> SAUTER A LA LIGNE (pas besoin de fermer)
  • < ol> < /ol> Ordered Liste (avec n°) + < li>< /li> Liste avec chiffre
  • < /ul> < /ul> Unordered Liste (avec des points) + < li>< /li> Liste avec points
  • < a href="Index.html">aller sur une autre page< /a>
  • < a href="Index.html" target="_blanc">ouvrir dans un nouvel onglet< /a>
  • Si la page est dans un sous dossier, il faut noter : dossier/index.html
  • Lorsqu'on nomme le chemin, et qu'on veut remonter vers un dossier superieur il faut noter ../ pour chaque dossier parent
  • Lorsque le site sera hébergé, les chemins doivent commencer par / pour indiquer la racine
  • on peut ancrer un mot par exemple "mot clé" dans une phrase < a href="#mot-cle">mot clé< /a>, lorsqu'on clique dessus, ça emmène vers le même mot ailleurs, il faut utiliser l'attribut id dans une balise (ex strong) : < strong id=mot-cle>mot clé

    • Commandes

  • Pour insérer le code HTML d'une page standard : taper html:5. La page est "coupée" en deux portions :
    1. L'en-tête
      • < !DOCTYPE html>
      • < html lang="fr">
      • < head>
      • < meta charset="UTF-8">
      • < title> écrire le titre de la page < /title>
      • < /head>

    2. Le corps
      • < body> Tout le code de la page va se trouver entre ces deux balises < /body>

    3. On ferme la page < /html>
