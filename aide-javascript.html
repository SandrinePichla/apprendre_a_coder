Qu'est-ce qu'une variable

En informatique, une variable est un conteneur qui stocke la donnée temporairement au sein de votre code. Cela vous permet d'enregistrer des données.

Valeur, nom, type

Une variable c'est un ensemble de 3 élements : une valeur (Sandrine), un nom (monPrenom), un type (du texte)

let ou const (ne jamais mettre de mascules)

  • Const : Une variable peut être fixe, par exemple mon prénom, une fois déclaré, mon prénom ne pourra plus changer. Si on essaie, JavaScript refusera d'exécuter l'instruction et affichera une erreur.
    On écrira : const monPrenom = "Sandrine"
  • Let : Elle peut changer : par exemple mon âge. On écrira : let monAge = 42
  • Var : L'instruction var peut également être utilisée pour déclarer une variable, mais elle est considérée comme obsolète.
  • L'instruction console.log() : Pour vérifier le contenu d'une variable, il est possible d'utiliser l'instruction console.log(), avec entre les parenthèses, le nom de la variable.
  • Il n'est pas necessaire de requalifier la variable quand elle a déjà été définie par let ou const.
  • Importance de nommer correctement les variables : il faut donner du sens pour qu'un autre développeur puisse comprendre

    • Exemple de code et vérification avec la console. Erreur sur la variable const

    Types de variables : number, string et boolean

  • number = nombre (âge, prix...)
  • String = chaine de caracères (prénom, texte...)
  • Boolean = true/false (seulement 2 possibilités)

    • Modifier les variables

  • Modifier la variable Number Ex d'une variable qui compte le nombre d'adhérents (200), on veut rajouter 100. On commence par :
    let V = 200
    V = V + 100
    ou V += 100
  • Modifier la variable String On veut concatener des variables texte. On commence par :
    let V1 = "Bienvenue, "
    let V2 = "Sandrine"
    let message = V1 + V2
    On peut aussi utiliser le signe + pour rajouter du texte
  • Modifier la variable Boolean (vrai/faux)
  • Pour aller plus loin : Autres opérateurs (de comparaison, arithmétique, de conditions...)
  • Les caractères sont entre " " : let variablestring = "Sandrine". Pour ne pas faire d'erreur sur les nombres (ne jamais les entourer de " "), on peut noter : let variableNumber = 150 ou let variableNumber = Number("150"). Pour les Booléans, pas de " " non plus : let variableBoolean = true

    Les objets JavaScript

    Un objet est un conteneur. Il est composé de propriétés qui ont chacune une valeur. Ainsi, le type de donnée Object offre la possibilité de stocker plusieurs valeurs en une seule fois, plutôt que de devoir stocker séparément nos valeurs dans plusieurs variables différentes.

    Comment déclarer un objet :

    Utiliser les accolades { }, les propriétés d'un objet ont un nom et une valeur qui sont assignées avec deux points, elles sont séparées par des virgules.Ex avec une personne :
    let monPersonnage = {
    nom: "Wayne",
    prenom: "Bruce",
    age: 35,
    couleurPreferee: "noir",
    hobbies: "sortir la nuit"
    }

    Pour rajouter une propriété : on utilise le . :

    monPersonnage.vehiculePrefere = "Batmobile"

    Acceder à une propriété d'un objet pour lui attribuer des variables :

    on utilise le . entre l'objet et la propriété. On peut aussi lui attribuer un nom plus court pour simplifier la recherche

    Opérateurs de comparaison

    • Inférieur ou égale à, se note <=
    • égal à, se note === .ATENTION, ne pas confondre avec = qui fixe une valeur :
        a = 42 signifie que l'on met la valeur 42 dans la variable a ;
        a === 42 signifie que l'on compare la valeur a et la valeur 42, pour savoir si ce sont les mêmes ou pas.
    • Différent de, se note !==

    La boucle FOR (=pour)

    • permet de répéter du code lorsque l'on sait d'avance combien de fois il faudra le répéter.
    • Par exemple, si nous voulons demander exactement trois fois à l'utilisateur d'entrer un mot.
    • for (let compteur = 0; compteur < 3; compteur++) { console.log(compteur)}

    La boucle WHILE (=tant que)

    • permet de répéter du code autant de fois qu'il le faut pour qu'une condition ne soit plus vraie.
    • Par exemple, si nous voulons redemander un mot à l'utilisateur jusqu'à ce que ce mot soit correct.
