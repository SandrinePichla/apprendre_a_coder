Raccourcis clavier

  • Ctrl C,Ctrl X, Ctrl V, Ctrl Z, Ctrl Y ...
  • Alt + flèche directionnelle
  • Rafraichir le navigateur Ctrl R

Commandes

  1. MENU HAUT (OPTIONS)

    • Ctrl Shift p : ouverture de la palette de commande, commandes récement utilisées
    • Ctrl , = ouvrir les paramètres
    • Ctrl k ctrl s = ouvre les raccourcis
    • Ctrl k ctr T = changement de thème

  2. MENU COTE (BARRE D'ACTIVITE)

    • Ctrl B ouvre / ferme le menu gauche
    • Ctrl shift F ♀recherhe dans tous les fichiers (menu à gauche), possibilité de remplacer
    • Ctrl shift E : explorateur
    • Ctrl shift X : extentions (plugin)
    • Ctrl shift D : déboguer
    • Ctrl shift G : GIT = console de sauvegarde sur Gitlab
    • Ctrl j ouvre / ferme le terminal
    • Ctrl p : console qui montre tous les fichiers

  3. BARRE EDITEUR DE FICHIERS

    • Ctrl tab = selection du fichier ouvert dans la console garder le ctrl et entrer
    • Ctrl W : ferme le fichier
    • Ctrl shift T ouvre le dernier fichier fermé

  4. DANS LE FICHIER

    • Alt + flèche directionnelle Pour déplacer du code, le selectionner dans la marge +
    • Touche tab dans la ligne de code : indente, lors de l'ouverture d'une balise : créé la balise ouverte + fermée

    • Remplacer du texte : ctrl +H = console de remplacement
    • Ctrl + R renomme un objet dans tout le fichier
    • Ctrl + alt + clic créé plusieurs curseurs
    • Ctrl F = recherche ; dans le fichier; F3 pour trouver les résultats
    • Alt flèche, déplace le texte
