Vérification

  1. Inspecteur : dans les outils de Developpement des navigateurs, cliquer sur inspecter. Permet de voir le code, de voir les erreurs de saisie.
    • Dans l'onglet Elements, on peut repérer les balises pas fermées, le navigateur va interpreter les erreurs, et par exemple fermer une balise si oublié. On peut modifier du code dans l'inspecteur pour voir ce que ça donne sur le web. Ça ne modifie pas dans l'éditeur.
    • Console : permet de voir les erreurs, les grosses erreurs
  2. vérifier la compatibilté du code avec les navigateurs : I can use
