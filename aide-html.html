  • <p> </p> PARAGRAPHE
  • <ol> </ol> Ordered Liste (avec n°) + <li> </li> Liste avec chiffre
  • <ul> </ul> Unordered Liste (avec des points) + <li> </li> Liste avec points
  • <div> </div>, sépare des blocs pour appliquer un style, tel que lang. Différents attributs sont possibles
    • Très importanytes avec le CSS, car permet d'utiliser l'attribut class dans une balise div pour imposer un style à une portion de texte
    • lang permet de dire qu'une partie est dans une langue et puis dans une autre
    • Attribut Class exemple <div class="bigText"></div> augmente la taille du texte après apllication du CSS
  • contrairement à la div, le span est utilisé pour quelques mots, dans une phrase, il ne sépare pas le texte. Utilisé avec span pour insérer du CSS
    • Attribut Class exemple <span class="bigWords">></span> affiche les mots en plus graos avec du CSS

  • <table> </table>, création d'un tableau. S'utilise avec <tr> </tr>pour créer des lignes et <td> </td>pour créer une céllule
  • Mettre en forme le tableau
  • Colspan est utilisé lorsque dans un tableau, il y moins de celulle sur la ligne du coup, il faut préciser sur combien de colonne cette cellule s'étend
  • Rowspan est l'homologue du colspan mais pour les lignes : fusion de X lignes
  • Exemple avec une image qui s'étend sur 2 cellules :<tr><td colspann=2><img src="img/furet-chocolat.jpg" alt="Furet Zibeline chocolat" width="150"></td></tr>
  • Un tableau peut être fait d'une en-tête, on encadre le tr et le td de <thead>, la liste du tableau est encadrées d'un <tbody> et la dernière ligne pourrait être un total par exemple, on peut l'encadrer d'un <tfoot>. On peut remplacer le td par un th lorsqu'il s'agit d'une cellule d'en-tête, mais ce n'est pas indispensable, c'est pour la compréhension.
  • Voir la page test
  • On peut également l'imbriquer avec iframe :(voir contenu embarqué ci-dessous)
  • Sans CSS, le tableau n'a aucune bordure. Pour rajouter les bordures, on peut coller dans l'en-tête de la page un style, et cette portion de code CSS (à voir dans le chapitre CSS) :
    table,
    td {
    border: 1px solid #333;
    }
    thead,
    tfoot {
    background-color: #333;
    color: #fff;
    }

    Chemin vers une page interne au site

  • Si la page est dans un sous dossier, il faut noter : dossier/index.html
  • Lorsqu'on nomme le chemin, et qu'on veut remonter vers un dossier superieur il faut noter ../ pour chaque dossier parent
  • Lorsque le site sera hébergé, les chemins doivent commencer par / pour indiquer la racine

    Commandes

  • Pour insérer le code HTML d'une page standard : taper html:5. La page est "coupée" en deux portions :
    1. L'en-tête
      • <!DOCTYPE html>
      • <html lang="fr">
      • <head>
      • <meta charset="UTF-8">
      • <title> écrire le titre de la page </title>
      • </head>

    2. Le corps
      • <body> Tout le code de la page va se trouver entre ces deux balises </body>

    3. On ferme la page </html>

    img Elément d'image embarquée

    • <src> indique le chemin vers l'image (comme href)
    • <alt> description de l'image
    • formats des images jpg et jpeg (photos), png (graphique,applat de couleur,dégradé simple, logos, supporte la transparence), svg (logos de forme simple seulement), gif (animé, mais peu utilisé)
    • <width>largeur en pixel
    • <height> hauteur en pixel
    • Pour ouvrir cette image dans un nouvel onglet, il faut l'encadrer d'un <a href=""agt; en notant le chemin de l'image
    • Exemple <a href="Img/zibeline.jpg" target="_blank"><img src="Img/zibeline.jpg" alt="Photos furet zibeline" width="150" height="100"></a>
